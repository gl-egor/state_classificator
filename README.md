# arXiv Topic Classifier

## Что делает
Классифицирует научные статьи по тематикам (cs, physics, math, stat, q-bio), основываясь на их названии и описании.

## Процесс работы над проектом
В проекте используется модель DistilBERT, которую я дообучил на датасете [https://www.kaggle.com/datasets/neelshah18/arxivdataset/data]. Решалась задача мультилейбл классификации по отобранным самым популярным тематикам с датасета. Далее я отобрал 10000 примеров статей обучил 5 эпох на них DistilBert. В результате были получены метрики accuracy = 0.71, f1_micro  = 0.88, f1_macro = 0.47.
